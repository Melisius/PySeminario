# PySeminario
![Logo](logo.png)


```
usage: make_ff.py [-h] -i ITP [-o OUT] --fchk FCHK
                  [--parameter-type {seminario,seminario_modified,hessian_fit}]

Generate force constants based on projections of the Hessian

optional arguments:
  -h, --help            show this help message and exit
  -i ITP, --itp ITP     Name of input .itp file
  -o OUT                Name of output parameter file
  --fchk FCHK           Name of formatted checkpoint file to put into output
                        .itp
  --parameter-type {seminario,seminario_modified,hessian_fit}
                        Parameter type to use
```
